# Science of Habits

Интерактивный дэшборд **80 ключевых научных источников** о привычках, гормонах счастья, поведении и нейронауках.

## 🌐 Открыть онлайн

**GitHub Pages:** [https://bakhytbekzhumanov-lab.github.io/science_of_habits/](https://bakhytbekzhumanov-lab.github.io/science_of_habits/)

## 📂 Структура

- `index.html` — основной дэшборд (React + Tailwind, standalone)
- `hooked-mindmap.html` — интерактивная mindmap книги "Hooked" by Nir Eyal

## 🔑 Ключевые цифры

- **66 дней** — среднее время формирования привычки (Lally 2010)
- **43%** — дневных действий автоматичны (Wood 2002)
- **+500%** — продуктивность в состоянии потока (Kotler/McKinsey)
- **50/10/40** — генетика/обстоятельства/действия в формуле счастья

## 🛠 Технологии

React 18 + Babel Standalone + Tailwind CSS CDN — без сборки, один HTML-файл.
