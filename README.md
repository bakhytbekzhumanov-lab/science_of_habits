# 🧠 Science of Habits

Интерактивный дэшборд **80 научных источников** о привычках, гормонах счастья, поведении и нейронауках.

## 🌐 Открыть онлайн

**[https://bakhytbekzhumanov-lab.github.io/science_of_habits/](https://bakhytbekzhumanov-lab.github.io/science_of_habits/)**

## 🗂 Структура

| Файл | Описание |
|------|----------|
| `index.html` | Основной дэшборд — 80 источников с фильтрами, поиском и подробными карточками |
| `mindmap-viewer.html` | Интерактивный просмотрщик mind map (pan/zoom/animated nodes) |
| `hooked-mindmap.html` | Отдельная mindmap книги "Hooked" by Nir Eyal |

## ✨ Возможности

### Список источников
- 80 книг и научных статей по 10 категориям
- Фильтры по категории, типу (книга/статья), поиск
- Сортировка по Impact Score, году, алфавиту

### Подробная карточка (клик на источник)
- 📋 Подробная выжимка (500+ слов для ключевых источников)
- 📊 Ключевые цифры и статистика
- 📈 Визуализации данных
- 🧠 SVG Mind Map концепций
- ✅ Практические выводы
- 🔗 Кнопка "Открыть интерактивную Mind Map"

### Интерактивные Mind Map
- Pan (перетаскивание) + Zoom (колёсико)
- Анимированные узлы с плавным появлением
- SVG кривые связи между узлами
- **15 кастомных детальных mindmap** для ключевых источников
- Авто-генерация mindmap для остальных источников

### Дэшборд
- Кривая формирования привычки (Lally 2010)
- Пирог счастья (Lyubomirsky 50/10/40)
- Таблица гормонов с триггерами/эффектами/рисками
- Ключевые цифры из исследований
